=======================================

Wwise Unity Integration

Audiokinetic Inc.
http://www.audiokinetic.com
=======================================


The Wwise Unity Integration is a Unity plugin plus API binding in Unity that allows the use of the Wwise sound engine in Unity games. It is referred to as the Integration in the rest of the document.

Before taking any actions, we suggest that you read the documentation available both under the Wwise/Documentation folder after importing the unitypackage, and on the Downloads page of our website. 

Note: After importing the .unitypackage, minimum steps are required to install the Integration. Read the documentation section "Install the integration in a Unity project" for additional details.
